Analysis of time-to-event endpoints (survival analysis)

STDLOGHR

Starting with a data set containing individual time-to-event data with right censoring and optional weights, this macro computes the standardized log hazard ratio, a measure of strength of association of the covariates with the risk of an event, from a multivariable Cox proportional hazards regression.
References:
Crager MR (2012). Generalizing the standardized hazard ratio to multivariate proportional hazards regression, with an application to clinical~genomic studies. Journal of Applied Statistics 39:399–417.
Crager MR (2020). Extensions of the absolute standardized hazard ratio and connections with measures of explained variation and variable importance. Lifetime Data Analysis 26:872–892.

STDLOGHR.sas      User guide

STDLOGHR_INT

Starting with a data set containing individual time-to-event data with right censoring, optional weights, and an indicator for randomized treatment, this macro computes the standardized log hazard ratio for interactions of treatment (or other randomized intervention) with a set of variables from a multivariable Cox proportional hazards regression.
Reference:
Crager MR (2020). Extensions of the absolute standardized hazard ratio and connections with measures of explained variation and variable importance. Lifetime Data Analysis 26:872–892.

Risk_Est_PH_reg_time_dep

This macro provides point and interval estimates of the risk of an event of interest using Cox proportional hazards regression. "External" time-dependent covariates, stratified cohort sampling designs, and left truncation of event times are accommodated.

Risk_Est_PH_reg_time_dep.sas      User guide

Risk_Est_PH_reg_comp_risk

This macro provides point and interval estimates of the absolute risk (value of the cumulative incidence function) of an event of interest in the presence of competing risks, using proportional regression of cause-specific hazards. Cohort sampling designs, time-dependent covariates and multiple censoring mechanisms are accommodated.
Reference:
Crager MR, Braun JV (2022). Interval estimation of the absolute risk of an event with competing risks using proportional regression of cause-specific hazards. American Journal of Applied Mathematics 10:59–85.

Analysis of binary (yes/no) endpoints

STDLOGOR

Starting with a data set containing individual binary outcome data and optional weights, this macro computes the standardized log odds ratio, a measure of strength of association with the probability of the outcome, from a multivariable logistic regression.
References:

Discovery analysis

ID_power

This macro calculates the identification power of features in a discovery analysis for association with some clinical outcome or state. Identification power is defined as the probability of identifying the feature as associated with outcome while controlling the false discovery rate (FDR).
Reference: Crager MR (2012). Prospective calculation of identification power for individual genes in analyses controlling the false discovery rate. Genetic Epidemiology 36:839–847.

ID_power.sas      User guide

TDRDAS

Starting with a vertical input file containing a list of predictors, their estimated standardized degrees of association with an outcome or state and the standard errors of the estimates, this macro identifies sets of the predictors with strong association with the outcome while controlling the false discovery rate (FDR) and produces estimates of the degree of association corrected for regression to the mean (RM).
Reference: Crager MR (2010). Gene identification using true discovery rate degree of association sets and estimates corrected for regression to the mean. Statistics in Medicine 29:33–45.

TDRDAS.sas      User guide

TDRDAS_SEPCLASS

Starting with a vertical input file containing a list of predictors, their estimated standardized degrees of association with an outcome or state, the standard errors of the estimates, and a class identifier separating the predictors into exhaustive, mutually exclusive classes, this macro identifies sets of the predictors with strong association with the outcome while controlling the false discovery rate (FDR) using Efron's separate class method applied to TDRDA analysis and produces estimates of the degree of association corrected for regression to the mean (RM) within class.
Reference: Crager MR, Ahmed M (2014). Separate Class True Discovery Rate Degree of Association Sets for Biomarker Identification. Journal of Biopharmaceutical Statistics 24:1022–1034.

Patient-specific meta-analysis (PSMA)

PSMA provides a method for estimating risk using meta-analysis methods applied to several studies. The PSMA calculation optimizes the use of the risk information provided by each study for patients like the one for whom the risk is being assessed based on their covariate values.

key_stats_phreg

This macro calculates key summary statistics from a Cox proportional hazards regression to be used in risk assessment with patient-specific meta-analysis.
Reference: Crager MR, Tang G (2014). Patient-specific meta-analysis for risk assessment using multivariate proportional hazards regression. Journal of Applied Statistics 41:2676–2695.

key_stats_phreg.sas      User guide

Risk_Est_PSMA_phreg

Starting with a key statistics data set produced for each of several studies using macro key_stats_phreg, this macro provides point and interval estimates of the risk of an event of interest using fixed-effects patient-specific meta-analysis.
Reference: Crager MR, Tang G (2014). Patient-specific meta-analysis for risk assessment using multivariate proportional hazards regression. Journal of Applied Statistics 41:2676–2695.
